# 🛒 Kirana Store Calculator

A simple Python-based billing system that simulates a real-world kirana (grocery) store. It handles customer transactions, applies discounts, and tracks total daily sales.

---

## 🚀 Features

- Add multiple products per customer
- Auto-generate unique Product IDs
- Apply discount (0–100%)
- Generate bill for each customer
- Track total sales for the entire day
- Input validation (name, price, discount)
- Handles edge cases:
  - No items entered
  - Stop during billing
  - Partial input handling

---

## 🧠 Concepts Used

- Loops (nested while loops)
- Conditional logic
- Dictionaries (data storage)
- String formatting (aligned output)
- Input validation

---

## ▶️ How to Run

1. Run the Python file:
   python main.py

2. Follow instructions:
   - Enter product name
   - Enter product price
   - Press Enter to generate bill
   - Type 'stop' to end program
