# 🎯 Number Guessing Game

A simple multi-player number guessing game built using Python.

## 🚀 Features
- Multiple players support
- Unique user IDs
- Input validation
- Limited attempts system
- Score calculation
- Displays highest scoring player(s)

## ▶️ How to run
Run the file using:
python main.py
